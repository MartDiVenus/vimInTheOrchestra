§2 tagbar 


*2.1 Senza il gestore/installer vim-plug
### installazione alternativa a git clone eseguita in ~/.vim
## installazione alternativa a vimball
## attenzione perché la cartella .github è necessario che si trovi in~/.vim,
## anche il file .vintrc.yaml è necessario che si trovi in ~/.vim

alearchiver -u -d. tagbar-master.zip

cd tagbar-master

## obbligatorio in  ~/.vim/autoload
cp -r autoload/* ~/.vim/autoload

cp plugin/* ~/.vim/plugin

mkdir ~/.vim/syntax 2> /dev/null

cp syntax/* ~/.vim/syntax

cp doc/* ~/.vim/doc

### e devi aggiornare il tag dell'help
vim -c ':helptags ~/.vim/doc' -c :q





*2.2 Ibrido: con il gestore/installer vim-plug ma offline

alearchiver -u -d. tagbar-master.zip

cd tagbar-master

## obbligatorio in  ~/.vim/autoload
cp -r autoload/* ~/.vim/autoload

mkdir -p ~/.vim/plugged/tagbar

cp -r plugin ~/.vim/plugged/tagbar

cp -r syntax ~/.vim/plugged/tagbar

cp doc/* ~/.vim/plugged/tagbar/doc




*2.3 Con il gestore/installer vim-plug, online

° I modo
call plug#begin('~/.vim/plugged')
Plug 'preservim/tagbar'
...
call plug#end()

°° I modo a)
:PlugInstall tagbar

°° I modo b) automaticamente magari da script
vim -c ":PlugInstall tagbar"
e alla fine dovrai digitare due volte <:q> <:q>

## obbligatorio in  ~/.vim/autoload
cp -r ~/.vim/plugged/tagbar/autoload/* ~/.vim/autoload



* II modo
o in Valentine.vim devi avere
call plug#begin('~/.vim/plugged')
...
Plug 'preservim/tagbar', { 'on': 'TagbarOpen', 'do': 'cp -r autoload/* ~/.vim/autoload' }
..
call plug#end()

°° II modo a)
:PlugInstall tagbar

°° II modo b) automaticamente magari da script
vim -c ":PlugInstall tagbar"
e alla fine dovrai digitare due volte <:q> <:q>
